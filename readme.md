# Avoska POS

## Что делает этот плагин?
Мощнейшая кассовая система (Point of Sale) для ручного пробития товаров продавцом в офлайн-режиме магазина или при приемке заказа по телефону. Интерфейс адаптирован под сканеры штрихкодов, содержит поиск по товарам, клиентам (добавление новых), расчету сдачи и созданию кастомных заказов.

## Как использовать
Создайте страницу "Касса" и добавьте шорткод `[avoska_pos]`. Рекомендуется открывать страницу на планшете или ПК. Приложение общается с WooCommerce через огромный набор AJAX-событий (создание заказов, изменение продуктов, получение сотрудников).

## Установка и активация
Установка производится активацией плагина `Avoska POS`. Дополнительных настроек не требует - сразу тянет всю базу товаров WooCommerce.
